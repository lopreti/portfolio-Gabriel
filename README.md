#  Portfólio - Gabriel Duarte

Este projeto é um portfólio pessoal desenvolvido para um desenvolvedor Back-End, com o objetivo de criar uma apresentação moderna, responsiva e profissional para suas habilidades, projetos e contato.

---

## 📌 Sobre o projeto

Este portfólio foi desenvolvido como uma ação de colaboração para ajudar um amigo desenvolvedor na construção de sua presença profissional online.

O foco principal foi criar uma interface visual impactante, com boa experiência de navegação, organização clara das informações e destaque para seus projetos e tecnologias.

---

## 🛠️ Tecnologias utilizadas

O projeto foi desenvolvido utilizando as seguintes tecnologias:

- HTML5
- CSS3
- JavaScript (Vanilla)
- Google Fonts (Montserrat)
- SVGs personalizados para ícones sociais
- Animações e efeitos visuais com JavaScript puro
- Canvas API (efeito de partículas no fundo)

---

## 🎨 Funcionalidades e recursos

- Layout moderno e responsivo
- Animações suaves de entrada (scroll e fade effects)
- Cursor personalizado interativo
- Efeito de partículas no fundo (Canvas)
- Navbar com navegação por âncoras
- Seção de projetos dinâmica e organizada
- Links diretos para GitHub, LinkedIn, Instagram e WhatsApp
- Design focado em experiência do usuário (UX/UI)
